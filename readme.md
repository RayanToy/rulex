## 📖 О проекте

**RuLex** — инновационная образовательная платформа, которая использует искусственный интеллект (Claude AI) для автоматической генерации тестовых заданий на проверку словарного запаса.

Платформа решает ключевую проблему: традиционные методы проверки лексики ограничены в точности и объективности. RuLex предлагает **адаптивное тестирование**, где сложность заданий подстраивается под уровень каждого ученика в реальном времени.

> [!NOTE]  
> Проект создан при поддержке **Фонда содействия инновациям** в рамках программы «Студенческий стартап» федерального проекта «Платформа университетского технологического предпринимательства».

## ✨ Возможности

### 🎓 Для учеников
- **Адаптивные тесты** — сложность автоматически подстраивается под уровень знаний.
- **Поддержка классов 3–11** и выпускников.
- **Подробные результаты** — разбивка по частотности слов (высоко-, средне-, низкочастотные).
- **Персональные рекомендации** по развитию словарного запаса.
- **История тестирований** — отслеживание прогресса.

### 🤖 AI-генерация вопросов
- Автоматическая генерация тестовых заданий через **Claude AI (Sonnet 4)**.
- Морфологический анализ слов через **pymorphy3**.
- Интеллектуальный подбор **дистракторов** (неправильных вариантов ответа).
- Проверка слов на пригодность для теста (фильтрация географических названий, этнонимов, узкоспециальных терминов).
- Автоматическая коррекция толкований (удаление однокоренных слов).

### 📊 Для преподавателей
- Генерация вопросов: одиночная, массовая и автоматическая.
- Ручное добавление и редактирование вопросов.
- База вопросов с фильтрацией.
- Частотные списки слов по классам.
